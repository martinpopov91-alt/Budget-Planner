INSTRUCTIONS OF USE

STEP 1: SET UP YOUR CURRENT BUDGET

Start by setting up the Currency, Start Date and End Date.

Proceed by populating the key financial data for the current Budget. This includes Categories and Budgeted Amounts for: Income, Bills, Expenses, Savings and Debt.

You can also have a Rollover Option if you wish.

IMPORTANT: Please note that the Tracker will show the exact transaction falling within the range between the Start and End Date.

STEP 2: RECORD TRANSACTIONS - ACTUALS

To record all your transactions please use the Transaction Tracker.

Please use all columns to record a Transaction - Date, Amount, Category, and Notes (Optional).

STEP 3: BONUS: GOAL TRACKER

Visualise your Goals, their distribution and monthly contribution targets.

STEP 4: GOAL SETTING

Start by writing the name of your goal (create a new "Savings" category), click the ‚öôÔ∏è icon.

Record your contributions using the Transaction Tracker (select "Savings" type).

Use the Date, Goal (Category) and Amount as Mandatory columns.

STEP 6: GOAL TRACKER DASHBOARD

Track your progress against the goal amount and your monthly contributions from the Transaction Tracker.
