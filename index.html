USER GUIDE & INSTRUCTIONS

1. GETTING STARTED

Settings Bar: Located at the top, this is your global control center.

  • Currency: Updates symbols ($, ‚Ç¨, ¬£, etc.) globally.
  • Dates: Define your budget period. "Days Left" and "Daily Budget" are calculated from the End Date.
  • Next Paycheck: Sets the countdown timer in the Dashboard.

2. TAB 1: BUDGET DASHBOARD (üìä)

This is your main overview. It shows exactly how much you have left to spend.

Blue Numbers (Budget): What you plan to spend.
White/Grey Numbers (Actual): What you actually spent. These update automatically when you add Transactions, but you can type over them to override.

Visual Report: Use the dropdown menu on the chart to switch between Overview (Totals) and Breakdowns (Specific Categories).

Goal Tracker: Located at the bottom. Click the ‚öôÔ∏è icon next to any "Savings" category to set a Target Goal (e.g., "New Car: $5,000") and a Start Balance.

3. TAB 2: TRANSACTION TRACKER (üìù)

Use this tab to log daily spending. Do not edit the dashboard tables directly for daily purchases.

  1. Select the Date and enter the Amount.
  2. Choose the Type (e.g., Expenses) and Category (e.g., Groceries).
  3. Click Add. The Budget Dashboard will update automatically.

4. TAB 3: DIVIDENDS & ASSETS (üíé)

Use this tab to track your Net Worth, Portfolio, and Passive Income.

  • üí∞ Buy Asset: Log this when you buy stocks or crypto. Increases "Total Invested".
  • üíé Dividend: Log this when you receive a payout. Increases "Dividends Collected".

5. CLOUD SYNC & DATA

Important: Data is saved to your browser automatically. If you clear your cache, data is lost.

To backup or sync between devices:

  1. Get a free GitHub Account.
  2. Generate a Personal Access Token (with 'gist' permission).
  3. Create a Public Gist and copy the ID.
